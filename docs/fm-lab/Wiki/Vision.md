# Vision

*One interface to rule them all — in your personal style of workflow:*

- Your FileMaker Solution
- Your Favorite Tools
- Your Agent
- Your Project Docs
- All FileMaker-related docs and knowledge
- All possible extensions
- All in one Interface

### What that means

FM-Lab has a well thought out architecture to grow over time.
There are some associated projects already in development (internal and external) that will integrate nicely with the provided setup and the ideas behind it.
On the other hand there will be integrations to existing tools that make your life easier during development.
